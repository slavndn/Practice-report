# Материалы к защите ВКР · «Школьный журнал ФК»

Документы выпускной квалификационной работы и производственной практики.

**Тема:** разработка веб-приложения для учёта спортивных достижений обучающихся на уроках физической культуры в школе.

**Продукт:** веб-приложение **«Школьный журнал ФК»** — журнал нормативов для 1–4 классов, автоматический расчёт баллов, рейтинги, турнирные сетки.

**Направление:** 09.03.01 «Информатика и вычислительная техника», профиль «Технологии разработки программного обеспечения».

**Вуз:** РГПУ им. А. И. Герцена · 2026
